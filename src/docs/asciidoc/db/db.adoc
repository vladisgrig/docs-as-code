= Структура данных сервиса работы с задачами

Структур данных, используемая сервисом работы с задачами, приведена на диаграмме.

[plantuml, db, png]
----
include::db.puml[]
----

.Задача
|===
|*Атрибут*  |*Ключ*  |*Тип*  |*Не пустой*  |*Описание*

|Заголовок
|title
|Однострочный текст
|true
|Краткое описание задачи.

Длинна не более 255 символов

|Описание
|description
|Многострочный текст
|false
|Полное описание задачи, что необходимо сделать
|===

.Статус
|===
|*Атрибут*  |*Ключ*  |*Тип*  |*Не пустой*  |*Описание*

|Наименование
|name
|Однострочный текст
|true
a|Статус исполнения задачи.

Одно из значений:

* `todo` - к исполнению;
* `in progress` - в работе;
* `done` - исполнена;
* `declined` - отменена.

По умолначнию `todo`
|===
