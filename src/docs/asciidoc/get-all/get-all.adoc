= Получение списка созданных задач (GET)

Метод получения списка созданных задач.

== Входные параметры

Входные параметры отсутствуют.

== Выходные параметры

Выходные параметры:

* в случае успеха вернуть список задач с параметрами, указанными в таблице;

* иначе вернуть описание ошибки.

.Параметры задач
|===
|*Параметр*  |*Ключ*  |*Тип*  |*Не пустой*  |*Описание*

|Идентификтор задачи
|task_id
|Число
|true
|Идентификатор задачи

|Заголовок задачи
|title
|Однострочный текст
|true
|Краткое описание задачи

|Описание
|description
|Многострочный текст
|false
|Полное описание задачи, что необходимо сделать

|Статус
|status
|Однострочный текст
|true
|Статус исполнения задачи.
|===
